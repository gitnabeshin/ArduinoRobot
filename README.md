# ArduinoRobot
This is my original Arduino robot control program

# Specs
- This robot has,
  - 2 blue LED eyes
  - 4 servos for eash arms
  - 1 speaker
- This robot can be controlled with,
  - Arduino Nano
  - PCA9685
